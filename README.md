# Cafe Review Analysis
Business Analysis minimal project based on sentiment analysis on review of a Cafe.

## Tech Used
- Frontend
  - Astro js (Partial Hydration Framework)
  - React, TypeScript (Plugins and integration)
  - Shadcn (UI Library)
  - Zed (IDE)
- Processing
  - Python (Language)
  - Tensorflow/Keras (DL Framework)
  - Sklearn + Pandas (Data Processing)
  - Google Colab (Virtual IDE)
- Data Visualization
  - Chart.js (For Frontend)
  - Matplotlib/Seaborn (For Processing)
- Backend (Future)
  - FastAPI (Async Web Framework)
